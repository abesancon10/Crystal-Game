Dog Show Entry Game

You will be given a random at the start of the game. This is the number of dogs that can participate in the dog show.
There are only four breeds showing: Shelties, Saluki's, Afghan Hounds, and Chinese Crested's. Each photo is of that breed and has been assigned a random number. By clicking on the photo your the number of dogs entered will add up.
To win you match the number of dogs entered to that of the max number of entries. If you enter more dogs into the show than permitted you lose.
Each time when the game starts, the game all the values will change.
